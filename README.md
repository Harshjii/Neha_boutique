# 👑 Neha Boutique — Luxury Indian Haute Couture & Bridal Atelier

A high-fashion luxury React e-commerce web application for **Neha Boutique**, featuring interactive bespoke bridal customization, 5-stage order production tracking, size guide fitting assistant, multi-step checkout, and seamless Instagram integration.

Official Instagram Handle: [**@neha_boutique_0**](https://www.instagram.com/neha_boutique_0/)

---

## 🌟 Key Features

- **Luxury Design Tokens:** Built with Google Fonts (`Playfair Display`, `Plus Jakarta Sans`, `Inter`), Jet Black (`#1a1a1a`), Metallic Accent Gold (`#c8a96a`), and glassmorphism backdrop-blur overlays.
- **Animated Page Transitions:** Powered by Framer Motion for smooth section reveals, drawer physics, and modal popups.
- **Interactive Custom Studio (`/custom-studio`):** Real-time swatch customizer for silhouettes, raw silk/velvet fabrics, Zardozi/Gota Patti embroidery motifs, and color swatches (Royal Crimson, Emerald Green, Champagne Gold, Ivory, Midnight Blue).
- **Multi-Step Checkout (`/checkout`):** Address input, signature wooden trunk box gift wrap, promo code discounts (`NEHAVIP10`), and credit card/deposit payment options.
- **5-Stage Order Production Tracking (`/order-confirmation`):** Real-time tracking timeline (Order Confirmed → Handcrafted by Karigars → Quality Inspection → VIP Courier Dispatch → Delivered) with printable invoices.
- **VIP Member Profile Dashboard (`/account`):** Order history, saved body measurements (Bust, Waist, Hips, Height), and address book.
- **Fit & Size Assistant (`/size-guide`):** Interactive unit converter (toggle between Inches and Centimeters) and 4-point measurement guide.
- **Global Trunk Shows & VIP Events (`/events`):** International event directory (London, Dubai, New York) with interactive RSVP modal.
- **Customer Care FAQ Accordions (`/faq`):** Searchable breakdown for shipping, alterations, silk mark authenticity, and care instructions.

## 🚀 Getting Started

### 1. Installation
```bash
git clone https://github.com/Harshjii/Neha_boutique.git
cd Neha_boutique
npm install
```

### 2. Development Server
```bash
npm run dev
```
Open `http://localhost:5173` in your browser.
